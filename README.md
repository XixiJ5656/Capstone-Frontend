     #xyz-ecommerce site:

     this is women clothing ecommerce size. Back End is deployed on Heroku, front end is deployed on firebase. Link is :https://xyz-ecommerce.web.app

     * Tech used in this app:

     Back end:
      * Spring Boot
      * Spring Security
      * PostgreSQL

     Front end:
     * react
     * axios
     * js-cookie
     * react-credit-cards
     * react-redux
     * react-router-dom
     * react-validation
     * redux-thunk
     * validator
